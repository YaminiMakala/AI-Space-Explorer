# AI Space Explorer

AI Space Explorer is a project that allows users to ask questions related to space and receive answers using natural language processing techniques. The project utilizes a dataset of space-related questions and answers and employs TF-IDF vectorization and cosine similarity to find the most relevant answer to a given question.
